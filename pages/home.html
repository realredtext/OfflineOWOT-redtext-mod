About OfflineOWOT

OfflineOWOT is an infinite grid of text editable by you at any time. The changes you make to the page are not saved unless you use the tile storage feature. This version is oriented towards scripters who wish to learn.

OfflineOWOT does not support other worlds and has a very poor account system. They all start off blank. But, that is all you need to start scripting.

This is an extension of FP's OfflineOWOT V3
